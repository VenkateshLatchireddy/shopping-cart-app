This is a React-based shopping cart application with a special **free gift** feature. When the cart total reaches ‚Çπ20000, a **Wireless Mouse** is automatically added as a free gift, and a **popup notification** appears.

## üöÄ Features
- Add, remove, and update product quantities in the cart.
- **Free Gift Unlock:** A free **Wireless Mouse** is added when the subtotal reaches ‚Çπ20000.
- **Popup Notification:** Displays a message when the free gift is unlocked.
- **Automatic Removal:** If the subtotal drops below ‚Çπ20000, the free gift is removed.
- Simple, clean UI with responsive design.
